Ipatimup - Gestão de Parque de Estacionamento

Aplicação para gerir a entrada e a saída de veículos de um parque de estacionamento. BackOffice de registo de matriculas de veículos. Impressão de cartões para uso dos veículos com os respetivo códigos.

Ipatimup – Stock Manager

Gestão de Stocks para material químico perigoso. O projeto consistiu em armazenar a entrada e a saída de produtos químicos por códigos de barras. Com esta aplicação poder-se-ia saber quais os produtos químicos perigosos que se encontravam dentro do instituto

Lipor – EcoShop

Aplicação desenvolvida para pessoas com deficiências motoras e visuais. O Ecoshop tem interação a partir de um ecrã tátil e de grande dimensão. A aplicação faz o registo do material que os clientes depositam nos ecopontos, oferecendo pontos acumuláveis para um cartão vale.

Unicer – Phone Web Application

A aplicação implementada tem a maior parte dos fluxos diretamente integrados com SAP. A Unicer passou a ter um front-end web para aprovação e consultas históricas com o ERP. A agilidade para aprovação ou rejeição de uma instancia, a partir de um dispositivo móvel, permite uma redução significativa no tempo de interação do fluxo de trabalho do utilizador. A aplicação permite a criação de relatórios e recolha de estatísticas sobre o tempo médio de cada fluxo, o que resulta numa melhoria contínua do fluxo de trabalho de negócios.
